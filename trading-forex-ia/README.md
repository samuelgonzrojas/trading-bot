# Trading Forex con IA

Repositorio para experimentar con trading algorítmico en Forex utilizando:
- Oanda API para datos y ejecución
- Backtrader para backtests
- IA (ML/DL) como módulo de predicción

## 🚀 Instalación
```bash
git clone <URL_DEL_REPO>
cd trading-forex-ia
python -m venv .venv
source .venv/bin/activate  # en Linux/Mac
.venv\Scripts\activate     # en Windows
pip install -r requirements.txt
